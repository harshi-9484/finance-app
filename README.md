# Personal Finance Companion App 💰

A simple and user-friendly finance tracking application built using Flutter.

---

## 🚀 Features

- 🔐 Login Screen (basic UI)
- 📊 Dashboard showing:
  - Current Balance
  - Total Income
  - Total Expenses
- ➕ Add Transactions (Income & Expense)
- 📋 Transaction List
- 📈 Insights Screen (financial summary)
- 🎯 Goal Tracker (savings progress)
- 🔄 Smooth navigation using bottom tabs

---

## 🛠️ Tech Stack

- Flutter
- Dart
- Stateful Widgets (for state management)

---

flutter pub get
